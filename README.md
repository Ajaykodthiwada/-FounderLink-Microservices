# FounderLink – Microservices-based Platform

This project is a microservices-based backend system built using Spring Boot.

## Features
- Microservices architecture
- Service discovery using Eureka Server
- API Gateway for routing requests
- Multiple services (Authentication, Investment, Notification)
- REST API-based communication
- Docker-based containerization

## Tech Stack
- Java
- Spring Boot
- Microservices
- Eureka Server
- API Gateway
- Docker
- REST APIs

## Description
FounderLink demonstrates how multiple backend services interact in a scalable and modular architecture using microservices.
